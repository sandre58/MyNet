# MyNet.Reflection

Cached reflection accessors, expression helpers, and object-graph utilities to reduce runtime reflection overhead.

[![MIT License](https://img.shields.io/github/license/sandre58/MyNet)](https://github.com/sandre58/MyNet/blob/main/LICENSE)
[![NuGet](https://img.shields.io/nuget/v/MyNet.Reflection)](https://www.nuget.org/packages/MyNet.Reflection)

**Target framework:** .NET 10

## Installation

```bash
dotnet add package MyNet.Reflection
```

## Quick start

Use the property cache and expression helpers from `MyNet.Reflection` when you need fast get/set by name without compiling expressions yourself each time. See `tests/MyNet.Reflection.Tests` for usage aligned with your scenario.
