## Features

| Feature | Description |
| :------ | :---------- |
| **Generators** | Culture-aware person, address, and internet data |
| **Stack** | Built on Generator and Geography packages |
| **Fixtures** | Realistic data for MVVM and API demos |
| **Cultures** | Integrates with Globalization settings |

---

## Installation

```bash
dotnet add package MyNet.Fakers
```

## Quick start

```csharp
// Explore MyNet.Fakers namespaces and tests/MyNet.Fakers.Tests
// for culture-aware person, address, and internet generators.
```

Pair with **MyNet.Globalization** culture settings for localized fake strings.
