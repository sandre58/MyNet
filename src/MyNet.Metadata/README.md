# MyNet.Metadata

Central metadata registry for types and members with pluggable feature providers for reflection-light, extensible APIs.

[![MIT License](https://img.shields.io/github/license/sandre58/MyNet)](https://github.com/sandre58/MyNet/blob/main/LICENSE)
[![NuGet](https://img.shields.io/nuget/v/MyNet.Metadata)](https://www.nuget.org/packages/MyNet.Metadata)

**Target framework:** .NET 10

## Installation

```bash
dotnet add package MyNet.Metadata
```

## Quick start

Register and resolve metadata features for types used by Observable and other subsystems. See the [metadata generation](https://github.com/sandre58/MyNet2/blob/main/docs/guides/observable.md#metadata-generation) section in the Observable guide.
