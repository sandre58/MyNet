# MyNet.Geography.Localization

Localized country display names (satellite resources) and MyNet.Humanizer formatters for geography types.

[![MIT License](https://img.shields.io/github/license/sandre58/MyNet)](https://github.com/sandre58/MyNet/blob/main/LICENSE)
[![NuGet](https://img.shields.io/nuget/v/MyNet.Geography.Localization)](https://www.nuget.org/packages/MyNet.Geography.Localization)

**Target framework:** .NET 10

## Installation

```bash
dotnet add package MyNet.Geography.Localization
```

## Quick start

Add the package when you need **translated country names** in UI or reports. Register Humanizer/geography formatters per [Geography guide](https://github.com/sandre58/MyNet/blob/main/docs/guides/geography.md).
