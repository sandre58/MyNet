## Features

| Feature | Description |
| :------ | :---------- |
| **Accessors** | Cached property and member accessors |
| **Expressions** | Expression-tree helpers for fast get/set |
| **Object graphs** | Traversal and inspection utilities |
| **Infrastructure** | Focused API for higher-level MyNet packages |

---

## Installation

```bash
dotnet add package MyNet.Reflection
```

## Quick start

Use the property cache and expression helpers from `MyNet.Reflection` when you need fast get/set by name without compiling expressions yourself each time. See `tests/MyNet.Reflection.Tests` for usage aligned with your scenario.
